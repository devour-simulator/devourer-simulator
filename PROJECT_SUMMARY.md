## 📖 使用说明

### 快速开始
1. 用浏览器打开 `index.html`
2. 选择你喜欢的角色
3. 使用 WASD 或方向键移动
4. 接近敌人自动战斗
5. 击杀敌人升级变强

### 系统要求
- 现代浏览器（Chrome 60+, Firefox 55+等）
- JavaScript 需要启用
- 本地存储空间 <1MB

## 🎓 技术实现

### 核心技术
- HTML5 Canvas 图形绘制
- Vanilla JavaScript 逻辑实现
- LocalStorage 数据持久化
- CSS3 样式和动画
- RequestAnimationFrame 游戏循环

### 主要类和函数
- `Character` - 角色基类
- `Enemy extends Character` - 敌人类
- `battle()` - 战斗系统
- `gameLoop()` - 主循环
- `checkCollisions()` - 碰撞检测
