# 🎮 吞噬模拟器 - 可爱治愈版

一个治愈系2D游戏，你将选择不同的动物角色，在开放地图中与其他生物战斗、升级、获得技能，最终达成最高分！

## 📋 游戏特点

### 🐾 丰富的角色选择
游戏提供6种初始可玩角色，各有独特属性：

| 角色 | 攻击 | 防御 | 速度 | HP | 特点 |
|------|------|------|------|----|----|
| 🐱 小猫 | 5 | 2 | 6 | 30 | 均衡型 |
| 🐰 小兔 | 3 | 1 | 8 | 25 | 速度型，脆皮 |
| 🦊 小狐狸 | 7 | 3 | 7 | 35 | 快速进攻手 |
| 🐻‍❄️ 黑熊 | 8 | 6 | 3 | 50 | 坦克型，防御强 |
| 🐯 老虎 | 10 | 5 | 4 | 45 | 强大进攻者 |
| 🦅 苍鹰 | 6 | 2 | 9 | 28 | 超高速度 |

### 🎯 战斗系统
- **回合制战斗**：靠近敌人自动触发战斗
- **属性计算**：速度影响先手权，防御减少伤害
- **战斗随机性**：每次伤害都有随机浮动，增加策略性

### ⭐ 升级与成长
- **经验系统**：击杀敌人获得经验（等级越高的敌人经验越多）
- **指数升级**：升级所需经验呈指数增长（1.15倍系数）
- **技能选择**：每次升级时从3个随机技能中选择一个

### 🛡️ 技能系统
共10种技能可供升级选择：

**常见技能**
- 💪 **强化拳击**：攻击+5
- 🛡️ **坚硬皮肤**：防御+4
- ⚡ **快速冲刺**：速度+3
- ❤️ **生命活力**：最大HP+10

**稀有技能**
- 🔥 **凶猛打击**：攻击+8
- 🏰 **铁壁防守**：防御+6
- 🌪️ **闪电速度**：速度+5
- 💚 **生命恢复**：最大HP+15

**传奇技能**
- ⚔️ **狂暴之力**：攻击+10 防御-2（高收益高风险）
- 🛡️ **终极守护**：防御+8 速度-1（坦克进阶）

### 🌍 开放地图
- **随机敌人**：地图上随机生成敌人
- **AI寻路**：敌人会随机移动和寻找目标
- **无缝游玩**：击杀一定数量敌人后，世界等级上升，敌人变强
- **网格背景**：治愈系美术风格

### 📊 统计系统
- **最高分记录**：游戏会记录本地最高击杀数
- **实时统计**：显示当前击杀数、世界等级、敌人数量

## 🎮 游戏操作

| 操作 | 按键 |
|------|------|
| 向上移动 | W 或 ↑ |
| 向下移动 | S 或 ↓ |
| 向左移动 | A 或 ← |
| 向右移动 | D 或 → |
| 选择角色 | 鼠标点击 |
| 选择技能 | 鼠标点击 |
| 重新开始 | 游戏结束后点击重新开始按钮 |

## 🎯 游戏目标

1. **选择角色**：根据喜好选择不同的动物角色
2. **击杀敌人**：在地图中寻找并击杀敌人获得经验
3. **升级成长**：通过升级选择技能来强化角色
4. **追求高分**：击杀更多敌人，创造最高纪录

## 💡 游戏建议

### 初心者推荐
- **小猫**：属性均衡，适合新手了解游戏机制
- **小狐狸**：攻防速度都不错，容易存活

### 高手挑战
- **小兔**：虽然脆皮，但超高速度可以先手击杀敌人
- **黑熊**：防御型角色，需要合理搭配技能

### 技能搭配策略
- **输出流**：多选攻击和伤害相关技能，快速秒杀敌人
- **防守流**：多选防御和HP技能，苟活直到反杀
- **速度流**：追求最高速度，利用先手优势

## 📝 使用指南

### 快速开始
1. 在浏览器中打开 `index.html`
2. 选择你喜欢的角色
3. 使用WASD或方向键移动
4. 接近敌人自动开始战斗
5. 击杀敌人获得经验并升级
6. 选择技能来强化你的角色

### 本地运行
```bash
# 简单方法：直接在浏览器中打开 index.html
# 或使用本地服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```
