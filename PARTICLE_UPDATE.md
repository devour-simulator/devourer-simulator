## 更新内容

已成功为游戏添加完整的粒子系统，增加了游戏的视觉反馈和玩法深度。

## 实现细节

### Particle 类
- **位置**：game.js 第270行
- **功能**：管理单个粒子的生命周期、物理、渲染
- **属性**：
  - 类型（经验/治疗）
  - 位置(x, y)
  - 速度(vx, vy)
  - 生命值(life)
  - 透明度变化

### 粒子类型
1. **⭐ 经验粒子**
   - 效果：+5经验
   - 颜色：金色
   - 概率：70%

2. **❤️ 治疗粒子**
   - 效果：+8生命值
   - 颜色：红色
   - 概率：30%

### 粒子生成
- **函数**：spawnParticles(x, y, count)
- **触发时机**：敌人被击杀时
- **生成数量**：每个敌人5个粒子
- **生成位置**：敌人死亡位置

### 粒子更新
- **位置更新**：每帧更新xy坐标
- **重力模拟**：y方向持续加速下降
- **生命衰减**：每帧生命值-1
- **透明衰减**：根据生命比例计算透明度

### 粒子渲染
- **使用Emoji**：⭐和❤️
- **大小变化**：根据生命值缩放
- **淡出效果**：透明度逐渐减弱
- **消失条件**：生命值<=0时移除

### 碰撞检测
- **检测对象**：玩家与所有粒子
- **检测方式**：圆形碰撞
- **收集效果**：
  - 经验粒子：+5经验
  - 治疗粒子：+8HP（不超过最大HP）
- **移除条件**：被收集时删除

## 代码集成

### gameState 扩展
```javascript
gameState.particles = []  // 粒子数组
```

### 游戏循环整合
1. **更新阶段**（gameLoop）
   - 更新所有粒子位置和状态
   - 移除过期粒子

2. **碰撞检测阶段**（checkCollisions）
   - 检测玩家与粒子碰撞
   - 应用粒子效果

3. **渲染阶段**（render）
   - 绘制所有活跃粒子

## 配置调整方法

如需调整粒子参数，编辑 game.js 中的 Particle 类：

```javascript
// 粒子寿命（帧数）
this.maxLife = 120;  // 改为其他数值

// 粒子初始速度
this.vx = (Math.random() - 0.5) * 3;     // 改变横向速度
this.vy = (Math.random() - 0.5) * 3 - 1; // 改变纵向速度

// 重力加速度（render中）
this.vy += 0.1;  // 改为其他数值
```

修改 spawnParticles 函数：
```javascript
// 生成数量
count = 5  // 改为其他数值

// 生成类型概率
Math.random() > 0.3 ? 'exp' : 'heal'  // 调整阈值改变概率
```
