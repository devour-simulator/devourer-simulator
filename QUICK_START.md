# 🎮 吞噬模拟器 - 快速上手指南

## 🚀 30秒快速开始

1. **打开游戏**
   - 用浏览器打开 `index.html`

2. **选择角色**
   - 点击你喜欢的动物卡片
   - 不同角色有不同强度：
     - 🐻 黑熊 - 坦克型（防御强）
     - 🐯 老虎 - 输出型（攻击强）
     - 🐰 小兔 - 速度型（速度强但脆）

3. **开始游戏**
   - WASD 或方向键移动
   - 靠近敌人自动战斗
   - 赢了获得经验

4. **升级变强**
   - 经验条满了升级
   - 选择技能增强属性
   - 技能越来越强力

5. **追求高分**
   - 击杀更多敌人
   - 创造最高纪录
   - 解锁隐藏角色

## ⭐ 技能系统

### 升级时可以选择的技能

**常见技能**（容易获得）
- 💪 **强化拳击** → 攻击 +5
- 🛡️ **坚硬皮肤** → 防御 +4
- ⚡ **快速冲刺** → 速度 +3
- ❤️ **生命活力** → 最大HP +10

**稀有技能**（偶尔出现，更强）
- 🔥 **凶猛打击** → 攻击 +8
- 🏰 **铁壁防守** → 防御 +6
- 🌪️ **闪电速度** → 速度 +5
- 💚 **生命恢复** → 最大HP +15

**传奇技能**（罕见，但很特殊）
- ⚔️ **狂暴之力** → 攻击 +10，防御 -2（高风险高收益）
- 🛡️ **终极守护** → 防御 +8，速度 -1（防守强化）

### 技能搭配策略

**输出流**（击杀快）
- 多选攻击技能
- 优势：快速秒杀
- 劣势：容易被反杀

**防守流**（活得长）
- 多选防御和HP技能
- 优势：生存能力强
- 劣势：击杀速度慢

**平衡流**（稳定输出）
- 攻防速都选
- 优势：适应多数情况
- 劣势：没有特别突出的优点

## 🏆 进阶技巧

### 1. 属性理解
- **攻击** - 越高伤害越多
- **防御** - 越高受伤越少（每0.5防御减1伤害）
- **速度** - 越高越容易先手攻击
- **HP** - 越高越能抗更多伤害

### 2. 战斗策略
- 遇到弱敌优先打（经验多）
- 血量少于对手时要小心
- 避开敌人群集区域
- 升级后血量会恢复

### 3. 技能选择
- 看剩余生存状况选择
- 血少选防御/HP
- 伤害不足选攻击
- 速度慢可考虑选速度

### 4. 长期生存
- 保持运动，别让敌人围住
- 及时升级提升属性
- 均衡属性更稳定
- 最高分很难，慢慢来

## 🔓 隐藏角色解锁

游戏中还有2个隐藏角色可以解锁！

**🐆 金钱豹**
- 需要击杀 **10个敌人**
- 属性：攻击9 防御4 速度8 HP40
- 特点：超强速度+不错的攻击

**🔥 火凤凰** 
- 需要击杀 **25个敌人**
- 属性：攻击12 防御7 速度5 HP55
- 特点：最强的初始角色，全能型

### 解锁技巧
- 选择容易存活的角色（黑熊、小猫）
- 寻找敌人多的地方
- 采取保守策略，活得越久击杀越多
- 一次游戏里多击杀几个就能解锁

## 📊 游戏界面说明

## 💡 常见问题

**Q: 为什么一开始就死了？**
A: 敌人很强，选择防御高的角色（黑熊）更容易存活

**Q: 怎样才能击杀很多敌人？**
A: 选择好角色组合技能，保持运动避免被包围

**Q: 隐藏角色真的能解锁吗？**
A: 能的！击杀够数量的敌人就会自动解锁

**Q: 游戏数据会保存吗？**
A: 会的！最高分和解锁进度都会自动保存

**Q: 可以玩很久吗？**
A: 理论上没有上限，越往后越难
