# 🌟 经验粒子系统使用指南

## 功能概述

游戏已成功添加了经验粒子系统！当你击杀敌人时，会在敌人死亡的位置散落多个粒子，你可以移动到粒子上吃掉它们来获得额外收益。

## 粒子类型

### ⭐ 经验粒子（星形）
- **外观**：闪闪发光的⭐星形
- **颜色**：金色（#FFD700）
- **效果**：+5经验值
- **概率**：约70%

### ❤️ 治疗粒子（心形）
- **外观**：红心❤️
- **颜色**：红色（#FF6B6B）
- **效果**：+8生命值
- **概率**：约30%

## 粒子生成规则

### 何时生成
- 敌人被你击杀时自动生成

### 生成数量
- 每个敌人死亡时生成 **5 个粒子**

### 生成位置
- 在敌人死亡的位置生成

### 粒子寿命
- 粒子存在时间：**120帧**（约2秒）
- 超过这个时间后粒子消失

## 粒子物理

### 初始速度
- 水平方向：随机 ±1.5 单位/帧
- 竖直方向：向上 ±1.5 单位/帧
- 方向完全随机，形成爆炸状散落

### 重力效应
- 粒子受重力影响，会逐渐下落
- 重力加速度：0.1 单位/帧²

### 消失效果
- 粒子逐渐透明（淡出效果）
- 粒子逐渐缩小
- 最后完全消失

## 粒子收集

### 如何收集
- 用你的角色靠近粒子即可自动收集
- 收集距离：与你的角色半径相同

### 收集效果
- **经验粒子**：立即获得 5 点经验
- **治疗粒子**：立即恢复 8 点生命值
- 收集后粒子消失

### 优先级
- 粒子收集是自动的，无需手动操作
- 离你越近的粒子越容易被收集

## 游戏策略

### 最大化收益
1. **追踪粒子**：击杀敌人后快速移动到粒子位置
2. **合理走位**：在击杀敌人时，尽量靠近敌人位置
3. **规划路线**：在多个敌人之间移动时，不要遗漏掉落的粒子

### 生存策略
- 当HP较低时，优先寻找❤️治疗粒子
- 治疗粒子可以帮助你撑过难关
- 合理利用粒子治疗，减少因失血而游戏结束的情况

### 升级加速
- 多收集⭐经验粒子可以加快升级速度
- 收集粒子的经验值加起来也很可观
- 对于快速升级来说很有帮助

## 视觉反馈

### 粒子显示
- 粒子使用大的emoji符号，易于识别
- ⭐ 星形明显，有金色光芒感
- ❤️ 心形明显，有治疗感

### 动画效果
- 粒子从敌人死亡点散落开去
- 粒子逐渐下落，模拟物理重力
- 粒子逐渐淡出消失，视觉舒适

### 收集反馈
- 粒子被收集后立即消失
- 对应的属性值在界面上立即更新
- 提供清晰的反馈

## 性能影响

### 优化设计
- 粒子数量有限制（每个敌人只生成5个）
- 粒子自动清理（寿命到期后移除）
- 不会导致内存泄漏

### 帧率影响
- 粒子系统对性能影响极小
- 即使同时有大量粒子也能保持60fps

## 技术实现

### 核心类
```javascript
class Particle {
    constructor(x, y, type = 'exp')  // 初始化粒子
    update()                           // 更新粒子位置和状态
    draw(ctx)                          // 绘制粒子
}
```

### 粒子管理
- 所有粒子存储在 `gameState.particles` 数组中
- 每帧自动更新粒子状态
- 生命值到期的粒子自动移除

### 碰撞检测
- 使用圆形碰撞检测
- 检测距离 = 玩家半径 + 粒子半径
- 碰撞检测每帧执行

## 调试信息

### 检查粒子系统
- 打开浏览器开发者工具（F12）
- 查看 `gameState.particles` 数组长度
- 确认粒子被正确生成和消除

### 常见问题

**Q: 为什么看不到粒子？**
- A: 粒子寿命很短（2秒），移动速度很快
- 多击杀几个敌人，会更容易看到粒子
- 在击杀敌人的位置停留，可以更容易收集粒子

**Q: 粒子为什么会消失？**
- A: 这是设计的一部分，粒子有2秒寿命
- 在这2秒内收集粒子就能获得收益
- 超过时间未收集则消失

**Q: 粒子对性能有影响吗？**
- A: 基本没有影响，系统已优化
- 粒子数量有限制，寿命有限制
- 性能表现与原版相同
